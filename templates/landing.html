Everything You Need

CoralCSS provides all the building blocks to create modern, accessible web applications.

Utility-First

Build complex designs with simple utility classes. No more fighting with CSS specificity.

JIT Compilation

Generate CSS on-demand with zero runtime overhead. Only ship the styles you actually use.

Headless Components

Fully accessible UI components with complete styling freedom. Works with any framework.

Dark Mode

Built-in dark mode support with automatic theme detection and smooth transitions.

Modern CSS

Container queries, anchor positioning, scroll-driven animations, and wide-gamut colors.

Flexible Layout

Powerful grid and flexbox utilities for building any layout you can imagine.
