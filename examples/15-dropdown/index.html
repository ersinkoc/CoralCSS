Dropdown Component

Accessible dropdown menus with full keyboard navigation (Arrow keys, Enter, Escape).

Basic Dropdown

Right-aligned Dropdown

Usage

import { createDropdown } from '@coral-css/core/components'

// Create dropdown
const dropdown = createDropdown('#my-dropdown')

// Open/close programmatically
dropdown.open()
dropdown.close()
dropdown.toggle()

// Listen for events
dropdown.element.addEventListener('coral:dropdown:select', (e) => {
  console.log('Selected:', e.detail.item)
})

// Keyboard navigation:
// - ArrowDown/ArrowUp: Navigate items
// - Enter/Space: Select item
// - Escape: Close dropdown
