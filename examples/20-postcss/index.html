PostCSS Integration

Use CoralCSS with PostCSS for flexible CSS processing pipelines.

1. Installation

npm install @coral-css/core postcss

2. Configure PostCSS

postcss.config.js

module.exports = {
  plugins: [
    require('@coral-css/core/postcss')({
      // Content paths for class extraction
      content: [
        './index.html',
        './src/**/*.{js,ts,jsx,tsx,vue,svelte}',
      ],

      // Dark mode strategy
      darkMode: 'class',

      // Optional: Preset to use
      preset: 'coral', // 'coral' | 'wind' | 'mini' | 'full'
    }),

    // Optional: Add autoprefixer after CoralCSS
    require('autoprefixer'),
  ],
}

3. Create CSS File

styles.css

/* Import CoralCSS layers */
@coral base;      /* Reset and base styles */
@coral theme;     /* CSS custom properties */
@coral utilities; /* Utility classes */
@coral components; /* Component styles */

/* Your custom styles go after */
.my-custom-class {
  /* Your styles */
}

Available Directives

Directive Description
@coral base CSS reset and base element styles
@coral theme CSS custom properties for colors, spacing, etc.
@coral utilities All utility classes
@coral components Component-specific styles
@apply Apply utility classes to a selector

Using @apply

styles.css

/* Create reusable component classes */
.btn {
  @apply px-4 py-2 font-medium rounded-lg transition;
}

.btn-primary {
  @apply bg-coral-500 text-white hover:bg-coral-600;
}

.btn-secondary {
  @apply bg-slate-100 text-slate-700 hover:bg-slate-200;
}

.card {
  @apply bg-white rounded-xl shadow-md p-6;
}

.input {
  @apply w-full px-3 py-2 border border-slate-300 rounded-lg
         focus:ring-2 focus:ring-coral-500 focus:border-coral-500;
}

Framework Integration

Next.js

// postcss.config.js (same config works!)
module.exports = {
  plugins: {
    '@coral-css/core/postcss': {
      content: ['./pages/**/*.{js,tsx}', './components/**/*.{js,tsx}'],
    },
  },
}

Nuxt

// nuxt.config.ts
export default defineNuxtConfig({
  postcss: {
    plugins: {
      '@coral-css/core/postcss': {
        content: ['./components/**/*.vue', './pages/**/*.vue'],
      },
    },
  },
})

Plugin Options

Option Type Description
content string[] Glob patterns for files containing classes
darkMode string Dark mode strategy: 'class', 'media', 'selector', 'auto'
preset string Preset to use: 'coral', 'wind', 'mini', 'full'
prefix string Optional prefix for all utility classes
