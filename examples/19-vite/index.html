Vite Integration

Use CoralCSS with Vite for optimized builds and hot module replacement.

1. Installation

npm install @coral-css/core

2. Configure Vite

vite.config.ts

import { defineConfig } from 'vite'
import coral from '@coral-css/core/vite'

export default defineConfig({
  plugins: [
    coral({
      // Dark mode strategy
      darkMode: 'class', // 'class' | 'media' | 'selector' | 'auto'

      // Optional: Custom prefix for classes
      prefix: '',

      // Optional: Content paths for extraction
      content: [
        './index.html',
        './src/**/*.{js,ts,jsx,tsx,vue,svelte}',
      ],
    }),
  ],
})

3. Import Virtual CSS

main.ts or App.tsx

// Import the virtual CSS module
import 'virtual:coral.css'

// Or in your HTML
// <link rel="stylesheet" href="virtual:coral.css">

4. Use Utility Classes

App.tsx

function App() {
  return (

Plugin Options

Option Type Default Description
darkMode string 'class' Dark mode strategy
prefix string '' Class prefix
content string[] ['**/*.html'] Files to scan for classes
preset string 'coral' Preset to use

Features

  • HMR Support - Hot module replacement for instant updates
  • Tree Shaking - Only includes used classes in production
  • Minification - Automatic CSS minification in production
  • Source Maps - Development source maps for debugging
